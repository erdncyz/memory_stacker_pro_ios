# GitHub Pages Setup

1. Bu repoyu GitHub'a push et.
2. GitHub > Repo > Settings > Pages
3. `Build and deployment` altinda:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/docs`
4. Kaydet.
5. Yayina alinan URL formati:
   - `https://<github-kullanici-adi>.github.io/<repo-adi>/`

## App Store Connect alanlari
- Support URL: `.../support.html`
- Marketing URL: `.../marketing.html`
- Privacy Policy URL: `.../privacy-policy.html`

## Not
- `support@example.com` e-posta adresini gercek destek adresinle degistir.
